# 🌍 Global Sales Revenue Analytics Dashboard

> An end-to-end Business Intelligence project built using **SQL (DuckDB), Python, Power BI, and DAX** to analyze global sales performance and generate actionable business insights.

---

##  Project Overview

This project analyzes the **Global Superstore Sales Dataset** to uncover key business insights related to sales, profitability, customer behavior, product performance, regional markets, and shipping efficiency.

The project follows a complete Data Analytics workflow:

- Data Cleaning using Python
- Exploratory Data Analysis using SQL (DuckDB)
- Interactive Dashboard Development in Power BI
- KPI Reporting using DAX
- Business Insight Generation

This dashboard enables decision-makers to monitor business performance through interactive visualizations and dynamic filtering.

# 📈 Key Business Insights

### Sales Performance

- Total Revenue exceeded **12.64 Million**
- Total Profit reached **1.47 Million**
- Profit Margin maintained at **11.62%**

---

### Market Insights

- APAC generated the highest revenue.
- Europe contributed strong profits.
- Canada had the smallest sales contribution.

---

### Product Insights

- Apple Smart Phone (Full Size) generated the highest sales.
- Canon imageCLASS 2200 Advanced Copier generated the highest profit.
- Cubify CubeX 3D Printer recorded the largest loss.

---

### Category Insights

Technology generated the highest sales.

Office Supplies produced the highest average profit margin.

Furniture generated comparatively lower profitability.

---

### Shipping Insights

Standard Class accounted for the highest number of orders.

Same Day shipping represented the smallest share of deliveries.

---

### Time Analysis

Sales increased consistently from 2011 to 2014.

The highest revenue was recorded during November and December.
